# Frontend Practice Projects

A growing collection of HTML and CSS exercises I'm building as I learn frontend development from the ground up. Each piece focuses on a specific concept — typography, buttons, product cards, banners, layout, links — so I can build real fluency in the fundamentals before moving on to JavaScript and React.

## 🔗 Live Demo

[View live](https://tito-osakwe.github.io/frontend-practice-projects/)

## 📖 About

This repo is part of my structured journey into frontend development. Rather than starting with one large project, I'm working through smaller, focused exercises — each one isolating a specific HTML or CSS concept so I can practice it deliberately. The goal is depth on the fundamentals: clean markup, predictable styling, responsive layouts, and an eye for clear, simple UI.

## 🎯 What's Inside

Hands-on practice across a range of common UI patterns, including:

- Typography and heading hierarchies
- Buttons and call-to-action styling
- Product card layouts (e-commerce style)
- Banners and promotional sections
- Links, navigation, and pagination
- Layout exercises with flexbox

## 🛠️ Built With

- HTML5
- CSS3

## ✨ What I'm Learning

- Writing semantic, well-structured HTML
- CSS fundamentals: the box model, selectors, specificity
- Layout with flexbox (and starting on grid)
- Designing for readability and visual hierarchy
- Translating real-world UI into clean code

## 🚀 What's Next

- Building a personal portfolio site from scratch
- Adding interactivity with JavaScript
- Practicing CSS Grid for more complex layouts
- Working with React
